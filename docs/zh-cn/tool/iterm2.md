# iterm2

> mac 端 终端神器


## 快捷键



### 标签

```
command + t 新建标签
command + n / 箭头 标签切换
command + enter 全屏
command + f
command + w
```

### 分屏

```
command + d
command + shift + d
command + [ / ] 切换
command + shift + h 剪切板历史
```


### 其他

```
ctrl  + a 行首
ctrl + e 行尾
ctrl + f 前进
ctrl + b 后退
ctrl + u 清除行
ctrl + p 上一条命令
ctrl + r 搜索历史
ctrl + d 删除光标字符
ctrl + h 删除光标之前
ctrl + w 删除光标之前的单词
ctrl + k 删除到文本末尾
command + r 清屏
ctrl + l 清屏
ctrl + t 交换光标文本
clear 清屏
```
