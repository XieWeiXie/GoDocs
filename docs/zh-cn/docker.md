## Docker


- Dockerfile

> 编写单个镜像的文件

- Docker command

> 操作 镜像、容器、服务等命令

- Docker compose

> 编写复杂镜像的文件


```
docker ps -a
docker ps -l
docker images
docker attach | ctrl + P | ctrl + Q
docker run -it -d alpine /bin/sh
docker exec -it *** /bin/sh
docker stop
docker kill
exit
docker top dockername



```
