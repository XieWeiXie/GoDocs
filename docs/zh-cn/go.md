# Golang


## 环境搭建


- GOROOT: 源代码的安装目录
- GOPATH: 项目的工程目录
  - src: 源代码
  - bin：编译后的文件
  - pkg：编译过程中生成的一些文件
## 基础概念


- package： 包里面的函数是如何调用的
- import: 路径的形式
