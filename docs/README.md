# 编程指南

- Git: 版本控制
- Markdown、Rst
- 数据库：pg、redis、mongodb
- Linux : 服务器端命令
- Shell : 脚本编写
- Go : 基础知识、实例
- docsify 文档
- Mac
- GraphQL
- RestfulAPI
